1. Was macht das System / Programm
----------------------------------
Das Programm dient dazu die in der Vorlesung entstandene
Bildbearbeitungssprache in Java zu realisieren. 
Ein PPL-Programm nach Java Cross-compiled kann durch die Virtuelle
Machine vearbeitet werden.




2. Wo finde ich die Dokus, sowie weiter Informationen
-----------------------------------------------------
Die Aufgabenstellung kann man unter 
    http://www.fh-wedel.de/~si/praktika/SoftwarePraktikum/PPL-VM.html
nachlesen.

Eine Zusammenfassung liegt dem Programm als LaTex und Pdf Dokument im
Verzeichnis "Documentation" bei.

Weiterhin liegt eine komplette Packetbeschreibung generiert durch
javadoc im selben Verzeichnis. Um die Dokumentation zu erstellen
einfach 

>make doc 

eintippen.



3. Welche Systemvoraussetzungen gibt es
----------------------------------------
Ein Javafähiger Rechner mit Java (Version 1.1.8 empfohlen) und einem
Arbeitsspeicher > 18 MB ist mindest Vorausetzung.





4. Welche zusätliche Software wird gebraucht
--------------------------------------------
Alle benötigte Software liegt dem Programm bei.
Es werden keine weiteren Quellen mehr gebraucht, um ein vorhandenes
PPL-Programm, daß als Java Datei vorliegt auszuführen.

Um ein PPL-Programm nach Java zu Compilieren, bitte Pdf/Latex-Dokument lesen.
